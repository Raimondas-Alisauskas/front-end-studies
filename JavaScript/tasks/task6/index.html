Sukurkite naują masyvą, kurio elementai būtų šalių masyvai (valstybės pavadinimas, gyv. skaičius, plotas). Valstybės plotas: 92212 km², plotas tenkantis vienam gyventojui: 8987.52 m². ``` p.s.: 1 km² = 1000000m²
